## Installation

Requirements:

- Git
- Python 3.8+ (if the project uses Python) or appropriate runtime for the code in this repo

Steps:

1. Clone the repository:

```bash
git clone https://github.com/spoudel26/CSP451-CheckPoint1-Sujana-Poudel.git
cd CSP451-CheckPoint1-Sujana-Poudel
```

2. (Optional) Create and activate a virtual environment (for Python projects):

```bash
python3 -m venv .venv
source .venv/bin/activate  # macOS / Linux
.venv\Scripts\activate     # Windows
```

3. Install dependencies (if any):

```bash
pip install -r requirements.txt
```

If this repository does not include a requirements.txt, follow the language-specific instructions included in the project files.
